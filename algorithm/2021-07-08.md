# 큐

- 기차터널과 같은 개념
- 맛집 대기줄과 같은 개념
- 스택은 입/출구가 1개, 큐는 입/출구가 다르다
- First In First Out(FIFO) = Last In Last Out(LILO)

### 큐 원리

- enQueue -----------------------------> deQueue
- front, rear 모두 -1이 기본값
- 원형큐는 -1로 하면 안된다.
  - 원형큐는 한칸을 못쓴다.(이유 : front와 rear가 빈건지 꽉찬건지 구분이 안된다.)
  - rear에서 +1 되면 front가 비어있는것이다.
